 IBM Concert Dataapps is an addon to IBM Concert product.
 It provides the ability to create actionable dashboards for GraphQL in a few easy
 steps. This allows you to manage and visualize information relevant to your business 
 in a custom build dashboard for your bussiness needs. 

 To install Concert Dataapps addon first IBM Concert needs to be installed.
 Follow the online IBM Concert documentation for exact installation steps.
